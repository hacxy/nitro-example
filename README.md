# Nitro Example

这是一个使用 [Nitro](https://nitro.unjs.io/guide#quick-start) 开发的后端应用案例

- [JsonWebToken](https://github.com/auth0/node-jsonwebtoken) 登录身份校验
- [zod](https://github.com/colinhacks/zod) 运行时类型校验并自动生成OpenAPI
- 统一响应参数格式
- 统一业务异常响应格式
- 文件上传示例
